HydroLang: An open-source web-based programming framework for hydrological sciences

Carlos Erazo Ramirez, Yusuf Serme, Ibrahim Demir

Título

HydroLang: An open-source web-based programming framework for hydrological sciences.

Autores

Carlos Erazo Ramirez, Yusuf Serme, Ibrahim Demir

Resumen

Este artículo presenta HydroLang, un marco de software modular, integrable y de código abierto diseñado para la investigación y educación en hidrología. HydroLang permite la recuperación, manipulación, análisis, visualización y modelado de datos relacionados con el agua en la web. Implementado en JavaScript, el marco está estructurado de manera modular para facilitar comandos encadenados y se puede utilizar en diversas plataformas web.

Palabras clave

HydroLang, investigación hidrológica, educación, análisis de datos, visualización geoespacial.

2022
